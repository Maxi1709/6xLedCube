# 6xLedCube
Ein 6x6x6 Led-Würfel mit einem Arduino Mega 2560 R3 bauen.

##Projektbeschreibung
Ein sechs mal sechs mal sechs Leds großer Led Würfel. Es werden einfarbige (blaue) Leds verwendet. Die Steuerung erfolgt direkt über die IO-Pins des Arduino, Schieberegister werden nicht benötigt.
